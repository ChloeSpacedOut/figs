HostAPI

A global API dedicated to specifically the host of the avatar. For other viewers, these do nothing.

Methods

getSlot(host: HostAPI, slot: String): ItemStack

Gets an ItemStack for the item in the given slot. Should use the same syntax for slots as the /item command.

sendChatMessage(host: HostAPI, text: String): nil

Sends the given message in the chat.

isFirstPerson(host: HostAPI): Boolean

Returns true if the camera is in first person mode.

setSubtitle(host: HostAPI, text: String): nil

Sets the current subtitle to the given text. The text is given as json.

setTitle(host: HostAPI, text: String): nil

Sets the current title to the given text. The text is given as json.

isHost(host: HostAPI): Boolean

Returns true if this instance of the script is running on host.

I'm going to be honest, I don't really know what this one does. It has something to do with the amount of time that titles stay on screen, but nothing more specific than that.

swingArm(host: HostAPI, offhand: Boolean): nil

Animates swinging the player's arm. If the boolean is true, then the offhand is the one that swings.

clearTitle(host: HostAPI): nil

Clears the current title from the GUI.

getTargetedEntity(host: HostAPI): Entity

Returns a proxy for your currently targeted entity. This entity appears on the F3 screen.

isCameraBackwards(host: HostAPI): Boolean

Returns true if the camera is facing backwards.

setActionbar(host: HostAPI, text: String): nil

Sets the action bar message to the given text. The boolean parameter defaults to false.

setActionbar(host: HostAPI, text: String, animated: Boolean): nil

Sets the action bar message to the given text. The boolean parameter defaults to false.
