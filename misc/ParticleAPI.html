ParticleAPI

A global API which is used for dealing with Minecraft's particles. Can currently only be used to summon a particle. Accessed using the name "particle".

Methods

addParticle(api: ParticleAPI, name: String, posVel: Vector6): nil addParticle(api: ParticleAPI, name: String, pos: Vector3): nil addParticle(api: ParticleAPI, name: String, pos: Vector3, vel: Vector3): nil addParticle(api: ParticleAPI, name: String, posX: Number, posY: Number, posZ: Number): nil addParticle(api: ParticleAPI, name: String, pos: Vector3, velX: Number, velY: Number, velZ: Number): nil addParticle(api: ParticleAPI, name: String, posX: Number, posY: Number, posZ: Number, vel: Vector3): nil addParticle(api: ParticleAPI, name: String, posX: Number, posY: Number, posZ: Number, velX: Number, velY: Number, velZ: Number): nil

Creates a particle with the given name at the specified position, with the given velocity. Some particles have special properties, like the "dust" particle. For these particles, the special properties can be put into the "name" parameter, the same way as it works for commands.
