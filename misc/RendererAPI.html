RendererAPI

A global API providing functions that change the way Minecraft renders your player.

Methods

canRenderFireOverlay(api: RendererAPI): Boolean

Gets whether or not fire should render on you when you're on fire. The default value is nil, meaning to use the vanilla default, which is true for players.

setRenderFireOverlay(api: RendererAPI, bool: Boolean): nil

Sets whether or not fire should render on you when you're on fire.

getShadowRadius(api: RendererAPI): Number

Gets the radius of your shadow. The default value is nil, which means to use the vanilla default of 0.5 for players.

setShadowRadius(api: RendererAPI, radius: Number): nil

Sets the radius of your shadow. The default value is nil, which means to use the vanilla default of 0.5 for players. The maximum value is 12.
