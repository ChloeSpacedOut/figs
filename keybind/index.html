keybind

KeybindAPI

A global API containing a function to create new Keybind instances.

Methods

create(api: KeybindAPI, name: String, key: String, gui: Boolean): Keybind

Creates and returns a new Keybind. The name is used in the keybind menu. The key parameter is an identifier for a key, such as "key.keyboard.r". The boolean gui indicates whether the keybind should always work, or if it should only work when you don't have a screen open.

Keybind

Represents a key binding for your script. Instances are obtained using the KeybindAPI's create() function.

Methods

getName(keybind: Keybind): String

Gets the name of the keybind, which you set when you created the keybind.

getKey(keybind: Keybind): String

Gets the current key for this keybind.

isDefault(keybind: Keybind): Boolean

Checks whether this key is currently set to its default state (not been changed using the keybind menu)

setKey(keybind: Keybind, key: String): nil

Sets the key for this keybind.

getKeyName(keybind: Keybind): String

Gets the name of the current key for this keybind.

isPressed(keybind: Keybind): Boolean

Gets whether this keybind is currently pressed down.

Fields

onPress: org.moon.figura.lua.types.LuaFunction

A function that runs when the key is pressed down.

onRelease: org.moon.figura.lua.types.LuaFunction

A function that runs when the key is released.
