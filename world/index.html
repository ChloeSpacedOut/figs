WorldAPI

A global API dedicated to reading information from the Minecraft world. Accessed using the name "world".

Methods

exists(): Boolean

Checks whether or not a world currently exists. This will almost always be true, but might be false on some occasions such as while travelling between dimensions.

getTime(): Number getTime(delta: Number): Number

Gets the current game time of the world. If delta is passed in, then it adds delta to the time. The default value of delta is zero.

getTimeOfDay(): Number getTimeOfDay(delta: Number): Number

Gets the current day time of the world. If delta is passed in, then it adds delta to the time. The default value of delta is zero.

getStrongRedstonePower(pos: Vector3): Integer getStrongRedstonePower(x: Number, y: Number, z: Number): Integer

Gets the direct redstone power level of the block at the given position.

getBiome(pos: Vector3): Biome getBiome(x: Number, y: Number, z: Number): Biome

Gets the Biome located at the given position.

getBlockState(pos: Vector3): BlockState getBlockState(x: Number, y: Number, z: Number): BlockState

Gets the BlockState of the block at the given position.

getRedstonePower(pos: Vector3): Integer getRedstonePower(x: Number, y: Number, z: Number): Integer

Gets the redstone power level of the block at the given position.

getBlockLightLevel(pos: Vector3): Integer getBlockLightLevel(x: Number, y: Number, z: Number): Integer

Gets the block light level of the block at the given position.

getRainGradient(): Number getRainGradient(delta: Number): Number

Gets the current rain gradient in the world, interpolated from the previous tick to the current one. The default value of delta is 1, which is the current tick.

getLightLevel(pos: Vector3): Integer getLightLevel(x: Number, y: Number, z: Number): Integer

Gets the overall light level of the block at the given position.

getSkyLightLevel(pos: Vector3): Integer getSkyLightLevel(x: Number, y: Number, z: Number): Integer

Gets the sky light level of the block at the given position.

isThundering(): Boolean

Gets whether or not there is currently thunder/lightning happening in the world.

getMoonPhase(): Integer

Gets the current moon phase of the world, stored as an integer.

isOpenSky(pos: Vector3): Boolean isOpenSky(x: Number, y: Number, z: Number): Boolean

Gets whether or not the sky is open at the given position.

BlockState

A proxy for a block state from Minecraft. Instances are obtained through the WorldAPI.

Methods

isOpaque(blockState: BlockState): Boolean

Gets whether or not this BlockState is opaque.

exists(blockState: BlockState): Boolean

Checks whether or not this BlockState currently exists. I can't think of a situation where it wouldn't exist, but this function is provided for you to check just in case.

hasEmissiveLighting(blockState: BlockState, pos: Vector3): Boolean hasEmissiveLighting(blockState: BlockState, x: Number, y: Number, z: Number): Boolean

Gets whether or not the BlockState, at this position, uses emissive lighting.

getComparatorOutput(blockState: BlockState, pos: Vector3): Integer getComparatorOutput(blockState: BlockState, x: Number, y: Number, z: Number): Integer

Gets the amount of signal strength a comparator would get from this BlockState.

getVelocityMultiplier(blockState: BlockState): Number

Gets the velocity multiplier of this BlockState. (Only Soul sand, honey blocks in vanilla)

getJumpVelocityMultiplier(blockState: BlockState): Number

Gets the jump velocity multiplier of this BlockState. (Literally just honey blocks in vanilla)

isFullCube(blockState: BlockState, pos: Vector3): Boolean isFullCube(blockState: BlockState, x: Number, y: Number, z: Number): Boolean

Gets whether or not the BlockState, at this position, has a full cube as its collision hitbox.

getHardness(blockState: BlockState, pos: Vector3): Number getHardness(blockState: BlockState, x: Number, y: Number, z: Number): Number

Gets the hardness of the BlockState, when placed at this position.

isTranslucent(blockState: BlockState, pos: Vector3): Boolean isTranslucent(blockState: BlockState, x: Number, y: Number, z: Number): Boolean

Gets whether or not the BlockState, at this position, would propagate sky light downwards.

getOpacity(blockState: BlockState, pos: Vector3): Integer getOpacity(blockState: BlockState, x: Number, y: Number, z: Number): Integer

Gets the opacity of the BlockState, in terms of how much it affects light levels.

getMapColor(blockState: BlockState, pos: Vector3): Vector3 getMapColor(blockState: BlockState, x: Number, y: Number, z: Number): Vector3

Gets the map color of this BlockState, as a Vector3 with R,G,B ranging 0 to 1.

isSolidBlock(blockState: BlockState, pos: Vector3): Boolean isSolidBlock(blockState: BlockState, x: Number, y: Number, z: Number): Boolean

Gets whether or not the BlockState, at this position, is a solid block.

getBlastResistance(blockState: BlockState): Number

Gets the blast resistance of this BlockState.

getLuminance(blockState: BlockState): Integer

Gets the emission light level of this BlockState.

asItem(blockState: BlockState): ItemStack

Returns an ItemStack representing this block in item form, whatever Minecraft deems that to be. If it cannot find an item for this block, it will return nil.

emitsRedstonePower(blockState: BlockState): Boolean

Gets whether or not this BlockState emits redstone power.

hasBlockEntity(blockState: BlockState): Boolean

Gets whether or not this BlockState has an associated block entity.

getFriction(blockState: BlockState): Number

Gets the friction of this BlockState. (Slime blocks and ice in vanilla)

Fields

readonly id: java.lang.String

The identifier of the block this BlockState comes from.

ItemStack

A proxy for an item stack from Minecraft.

Methods

exists(itemStack: ItemStack): Boolean

Checks whether or not this ItemStack currently exists. Minecraft may delete ItemStack instances on occasion. I don't think it will delete them while they're in someone's inventory or on the ground, but testing is required to know for sure. Contact devs once someone has an answer to when these stop existing!

Fields

readonly id: java.lang.String

Contains the id of the item this ItemStack refers to.
